Moving Image Map Areas

After you have defined an area within an image map, you can move that area somewhere else within the image:

To move the area directly from the keyboard

  1. Move the cursor into an area of the image map.
  2. Press the Control key and click the mouse. This will highlight the image map area.
  3. Release the Control key but continue to hold down the mouse button. Use the mouse to move the highlighted image area to its new location.

To move the area using the Structure view

  1. Select Show Structure from the Views menu (or press Alt + v then Alt + s). The structure view of the document will open in another window.
  2. Edit the coords attribute manually. Example: coords=28,5,55,33.
  3. The image map area will move accordingly in the open browser window.
