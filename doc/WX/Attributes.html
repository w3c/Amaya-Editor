Editing Attributes

You can edit attributes in two ways:

  • Using the Attributes tool.
  • Directly in the Structure view.

Editing Attributes Through the Attributes Tool

The Attributes tool displays all attributes that can be associated with the selected element. It can be used to create an attribute, change the value of an existing attribute, or delete an attribute.

To create an attribute:

  1. Click within the element and press the F2 key to select the whole element.
  2. When the element is selected, you click on the [ ] before the name of the attribute you want to add. This attribute is now activated [x].
  3. For boolean attributes (such as ismap for an image), the command is finished.
  4. For other attributes you have to edit the value, then confirm with the apply button ().

To edit an attribute:

  1. Click within the element and press the F2 key to select the whole element.
  2. Once the element is selected with this active attribute (the name is preceded by a [x]), click on the entry name to display its current value.
  3. For attributes with a limited set of values (such as align for a paragraph or a heading), Amaya displays:
    • A set of radio buttons proposing the list of possible values. A long list (such as lang) is presented with a combo box.
    • An apply button () to confirm the selected value.
    • A trash button () to remove the attribute.
  4. For attributes with a free value (such as href for an anchor), Amaya displays: style=""
    • An input box to insert the value.
    • An apply button () to confirm the inserted value.
    • A trash button () to remove the attribute.

To remove an attribute:

  1. Click within the element and press the F2 key to select the whole element.
  2. When the element is selected, you click on the [x] before the entry to remove the attribute.
  3. You can also select the attribute and click on the trash button ().

Editing Attributes in the Structure View

All attributes are displayed in the Structure view. Attributes may be displayed in blue or pink. When the attribute value is displayed in blue, you can only change it with the Attributes tool. This ensures that only one of the correct values is chosen.

Attributes with a free value can be edited in the Structure View.

  • You can click within the attribute value (displayed in pink) and edit the value like any other character string.
  • To select the whole value, you can click within the value then hit the F2 key.
  • To remove the attribute, you can select the whole value, then press the Delete key twice. The first Delete will remove the attribute content and the second will remove the attribute.

Copying Attributes

An attribute is not the same thing as element and an attribute cannot be copied by the Copy/Paste options on the Edit menu.

To copy an attribute, you must use the Attributes tool:

  1. Select the element with the attribute you want to copy.
  2. Select the appropriate attribute entry from the Attributes tool.
  3. The select the new element to which you want to associate the current attribute.
  4. Click the apply button ().
