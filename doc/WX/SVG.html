Amaya

Editing SVG graphics

Amaya enables you to include vector graphics in Web pages, following the Scalable Vector Graphics (SVG) specification.

Amaya handles SVG elements as structured components, in the same way as HTML elements. Therefore, you can manipulate SVG graphics in the same way you manipulate other parts of HTML documents.

All editing commands provided by Amaya for handling text are also available for handling graphics (for example it's possible to create a link on a SVG element).

Warning: SVG authoring is under development. You may find bugs not fixed yet and the user interface is likely to change in the future.

Creating Graphics with the Palette

To create a new SVG element in a document, move the insertion point to the position where you want to insert it, and then click on a button of the SVG panel.

The available graphics elements are:

  • connectors: line, simple or double arrow, zigzag.
  • text or embedded HTML fragment (foreignObject) and image.
  • basic shapes: square, rectangle, diamond, parallelogram, trapezium, triangles, circle, ellipse.
  • polygon, polyline, open and close Bézier curves.
  • 3D objects: cube, parallelepided, cylinder.

In formatted view, the creation of graphics requires direct interaction:

  • for connectors, basic shapes or 3D objects: you first select the position of the left corner by pressing the left mouse button. While holding down the button, move the mouse to the position of the opposite corner, then release the button.
  • for text and foreignObject: click on the position where you want to insert the element.
  • for polygon and polyline: enter each point by clicking the corresponding position with the left mouse button. To indicate the last point, use a double-click.
  • Bézier curves: same as polygons except that you need to indicate the direction of the tangents.

Applying transformations to graphics

To apply transformations on one object, you must first select it. The rectangular selection is useful to select several objects. The available transformations are:

  • group/ungroup objects
  • changing the level of objects: send to back, one level down, one level up, bring to front
  • rotate, rotate 90°.
  • vertical and horizontal symmetry
  • alignments: left, center, right, top, middle, bottom. If only one object is selected, the alignment is made according to the SVG canvas.

Styling graphics

You can change the color of the graphics using the Style panel:

  • background color is used to fill the graphics.
  • text color is used for the stroke color.
